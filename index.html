css-for-line-clamp-ellipsis

Here's the CSS required to make a paragraph truncate with an ellipsis after a specific number of lines (the vendor prefixes are required):

.line-clamp {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 5;
  overflow: hidden;
}
